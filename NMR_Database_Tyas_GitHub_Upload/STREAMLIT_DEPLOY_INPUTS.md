# Streamlit Cloud: Pilih Mana dan Isi Apa

## Saat klik `Create app`

Isi seperti ini:

1. `Repository`
   pilih repo GitHub kamu yang berisi project ini
   contoh:
   `username-github-kamu/naturalproduct-spectraldatabase`

2. `Branch`
   pilih:
   `main`

3. `Main file path`
   isi:
   `scripts/app.py`

4. `App URL`
   isi:
   `naturalproduct-spectraldatabase`

5. `Advanced settings`
   buka bagian ini

6. `Secrets`
   paste isi dari file:
   `STREAMLIT_SECRETS_TEMPLATE.toml`

## Jika ada pilihan Python version

Biarkan default dulu.

## Jika subdomain tidak tersedia

Coba salah satu:

1. `naturalproduct-spectraldatabase-tyas`
2. `np-spectraldatabase-tyas`
3. `naturalproductspectraldatabase`

## Setelah klik Deploy

Tunggu sampai app selesai build.

Kalau ada error:

1. jangan panik
2. copy pesan errornya
3. kirim ke saya
4. saya bantu perbaiki langkah berikutnya
