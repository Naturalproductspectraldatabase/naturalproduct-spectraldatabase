Natural Product Spectral Database: Step-by-Step Deploy Guide

Dokumen ini dibuat supaya kamu bisa mengikuti langkah deploy dengan pelan, jelas, dan minim istilah teknis.

Target URL yang disarankan:
https://naturalproduct-spectraldatabase.streamlit.app

Bagian 1. Yang Sudah Kamu Punya

  1. Akun GitHub
  2. Akun Streamlit Community Cloud
  3. Akun Supabase

Bagian 2. Repository GitHub

  1. Masuk ke GitHub.
  2. Klik New repository.
  3. Nama repo: naturalproduct-spectraldatabase.
  4. Pilih Public.
  5. Klik Create repository.
Jangan upload file ini ke GitHub:
.streamlit/secrets.toml

Bagian 3. File yang Perlu Diupload ke GitHub

Upload isi folder project Desktop/NMR_Database_Tyas ke repo.

Bagian 4. Buat Database Supabase

  1. Masuk ke Supabase.
  2. Klik New project.
  3. Setelah project jadi, buka SQL Editor.
  4. Buka file lokal database/supabase_schema.sql.
  5. Copy seluruh isinya.
  6. Paste ke SQL Editor.
  7. Klik Run.

Bagian 5. Import Data Lama ke Supabase

File CSV sudah disiapkan di:

/Users/triandatyas/Desktop/NMR_Database_Tyas/data/exports/supabase_import/

Import dengan urutan ini:

  1. compounds.csv
  2. proton_nmr.csv
  3. carbon_nmr.csv
  4. spectra_files.csv

Bagian 6. Deploy di Streamlit Cloud

Saat klik Create app, isi seperti ini:

  1. Repository: repo GitHub kamu
  2. Branch: main
  3. Main file path: scripts/app.py
  4. App URL: naturalproduct-spectraldatabase

Bagian 7. Isi Secrets

Paste ini di Advanced settings > Secrets:

NPDB_APPROVED_PASSWORD = "OnnamideA13."
NPDB_APPROVED_NAMES = ["tyas", "sensei"]

Lalu tambahkan nilai Supabase setelah tersedia:

SUPABASE_URL = "https://your-project.supabase.co"
SUPABASE_ANON_KEY = "your-anon-key"

Bagian 8. Login User

Format login:

  1. Username: npdb_tyas
  2. Username: npdb_sensei
  3. Password: OnnamideA13.

Bagian 9. Backup Lokal

Kalau hosting bermasalah, jalankan app lokal:

streamlit run /Users/triandatyas/Desktop/NMR_Database_Tyas/scripts/app.py

Lalu buka dari HP di Wi-Fi yang sama:

http://IP_LAPTOP:8501

Bagian 10. Cara Menjadi PDF

  1. Buka file HTML ini di browser.
  2. Tekan Cmd + P.
  3. Pilih Save as PDF.

Bagian 11. Cara Membuka di Word

  1. Buka file HTML ini dengan Microsoft Word.
  2. Word biasanya bisa membaca format HTML.
  3. Lalu simpan sebagai dokumen Word jika kamu mau.
Jika ada satu layar yang membuatmu ragu, berhenti di sana dan kirimkan screenshot atau tulisan yang muncul. Saya lanjutkan dari titik itu.
